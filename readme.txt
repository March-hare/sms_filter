== Description ==
Adds the ability to extend the functionality of filtering Ushahidi alerts based 
on user defined preferences.  To achieve this some of the core alerting 
functionality is replaced and extended.  

== Installation ==
1. Copy the entire /sms_filter/ directory into your /plugins/ directory.
2. Activate the plugin.
